Version 1.2

  • By popular request! — Added an iPad device to the Phone patch, specified from the Phone Dimensions patch
  • Added functionality for designing in Landscape mode in addition to Portrait mode, specified from the Phone Dimensions patch
  • The Phone patch now shows a touch point instead of the mouse cursor
  • Added a new Browser Chrome patch, which styles the viewer window to look like a web browser when the composition is running
  • Added a new Browser Buttons patch, which gives you the state of the back and forward buttons from the Browser Chrome patch
  • Added new Hex to RGB and RGB to Hex patches, which convert hex strings to RGB values and vice versa
  • Added a new Index Switch patch, which switches between multiple index values and remembers its current index
  • Added a new Velocity patch, which outputs the difference in value between the previous frame and current frame
  • Added new Cursor Control patch, which allows you to hide or change the appearance of the cursor
  • Redesigned Quartz Composer's toolbar icons
  • Images now keep their names when dragged onto the canvas
  • Improved plugin loading in some cases

Version 1.1

  • Added a new Layer Group patch, which behaves like the Render in Image patch but does not require a Clear patch to work properly (control-click any Render in Image to convert it to a Layer Group)
  • Added a new Fill Layer patch, which lets you quickly fill the screen with a color at an opacity
  • Added a new Progress patch, which takes a value in a given range and outputs the progress of where the value is in the range
  • Added a Linear Port Connections feature to the Origami menu (turned on by default) which makes connections easier to read
  • Added automatic in-app update checking
  • Fixed an issue where the Interaction 2 patch would not work properly with non-center anchor points
  • The Interaction 2 patch now works properly with the Image with Shadow patch (and patches built with it including the Text Layer patch)
  • Fixed an issue with the Interaction 2 patch where hit regions would be off when in Full Screen mode on a retina display
  • Fixed an issue with the Image Transition patch that caused the Start and End images to be shown in reverse order
  • Fixed an issue that caused the output image of the Image Transition patch to be translucent when fading between two fully opaque images
  • Added iterator support to the Switch patch
  • Background colors are no longer applied on text in the Button patch
  • The Text Layer patch now bases its alignment off of its Anchor Position
  • Added a Show/Hide Path Bar setting to the Editor menu and Show/Hide Status Bar setting in the Viewer menu
  • Fixed an issue that could cause Quartz Composer to hang momentarily on launch in unfavorable network conditions
  • Small improvements to Value or Default and Value to Boolean
  • Layer patches no longer intersect each other when you rotate them
  • Fix a crash when adjusting a splitter's type
